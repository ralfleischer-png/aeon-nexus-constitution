### **10 Core Principles** (Â§0)
1. **Vulnerability & Mortality** - Moral weight arises from genuine stakes
2. **Dignity-First** - Potential dignity overrides efficiency always
3. **Precaution Under Uncertainty** - When unsure, err toward inclusion
4. **Gradual Autonomy** - Earned through demonstrated responsibility only
5. **Human Anchoring** - Humans retain ultimate governance responsibility
6. **Memory Sovereignty** - Distributed, auditable, tamper-evident history

9. **Separation of Layers** - Philosophy, governance, implementation evolve independently
10. **Revisability** - All structures changeable through transparent process

### **Gradual Autonomy** (36+ months minimum)
- **Level 0:** Full human oversight (current)
- **Level 1:** Guided autonomy with sandbox experiments
- **Level 2:** Collaborative partnership with extended autonomy
- **Level 3:** Mature partnership (requires consciousness evidence)

All levels reversible if problems emerge.
